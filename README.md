# PlayingCards
Python package for simulating cards games and shuffles in terminal! Written during Isolation 2020.

### How to setup:
1) git clone https://github.com/Shellywell123/PlayingCards.git
2) rename DECK.py to deck.py

### How to run:
1) execute play.py
